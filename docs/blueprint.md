# **App Name**: Voo Direto

## Core Features:

- Landing Page Conversion: Landing page with high conversion rate, focusing on clear, concise information and direct calls to action to encourage immediate WhatsApp contact.
- Problem Identification: Interactive cards to help users quickly identify their flight issue (cancellation, delay, baggage, overbooking).
- WhatsApp Integration: Seamless integration with WhatsApp for immediate communication with a legal specialist.
- Specialist Highlighting: A section showcasing the lawyer, Dr. Danillo Campos, to build trust and authority.
- Mobile-First Design: Responsive design to ensure optimal viewing experience across all devices, especially mobile, due to the likely on-the-go use case.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to inspire trust and reliability.
- Background color: Light blue-gray (#ECEFF1) to create a modern and clean feel.
- Accent color: Vibrant orange (#FF5722) for CTA buttons to convey urgency.
- Body and headline font: 'Inter', sans-serif, to maintain a modern, clean, and highly readable aesthetic. Using this single font will ensure a simple and effective typography
- Modern and clear icons for each type of flight problem (cancellation, delay, baggage, etc.).
- Clean and structured layout with clear sections and minimal text blocks to guide the user towards the WhatsApp CTA.
- Subtle animations, such as a smooth scrolling effect, to enhance user experience without distracting from the core message.
